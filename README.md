# 🎮 Pong Game

A classic Pong game built with HTML, CSS, and JavaScript with beautiful pastel colors and an adjustable AI opponent.

## 🎨 Features

- **Beautiful Pastel Design** - Soft yellow and cream color scheme with elegant styling
- **Two Control Methods** - Move your paddle with your mouse Y-position or arrow keys (↑/↓)
- **AI Opponent** - Computer-controlled opponent with strategic difficulty
- **Physics-Based Gameplay** - Realistic ball bouncing with collision detection
- **Scoreboard** - Real-time score tracking with first-to-5 wins
- **Responsive Design** - Works perfectly on desktop and mobile devices
- **Glowing Effects** - Beautiful shadow effects on paddles, ball, and UI elements

## 🕹️ How to Play

1. **Open the Game** - Simply open `index.html` in your web browser
2. **Controls:**
   - Move your paddle (left side) using:
     - **Mouse:** Move your cursor up and down
     - **Keyboard:** Use Arrow Keys (↑/↓) or W/S keys
3. **Objective:** Bounce the ball past the AI to score points
4. **Winning:** First player to reach 5 points wins!

## 🎯 Game Mechanics

- **Ball Physics:** The ball bounces off walls and paddles with realistic physics
- **Spin System:** Where you hit the ball on your paddle affects its trajectory
- **Speed Increase:** Ball accelerates slightly with each paddle hit for dynamic gameplay
- **AI with Reaction Delay:** AI opponent has intentional reaction delays to make the game challenging but beatable

## 🎮 Game Controls

| Control | Action |
|---------|--------|
| Mouse Movement | Move paddle up/down |
| Arrow Keys (↑/↓) | Move paddle up/down |
| W/S Keys | Move paddle up/down |
| Start Game Button | Start or pause the game |
| Reset Score Button | Reset score to 0-0 |
