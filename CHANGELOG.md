# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-07-19

### Added
- Beautiful pastel yellow and cream color scheme
- Easier AI difficulty with reduced speed and reaction delays
- Responsive design with mobile support
- Enhanced visual effects with glowing shadows
- Comprehensive documentation (README, CONTRIBUTING, CHANGELOG)

### Changed
- AI speed reduced from 5 to 3 pixels per frame
- AI now has reaction delay system (5-20 frame random delays)
- AI detection range increased from 35 to 50 pixels for easier gameplay
- Improved button hover effects with gradient background
- Better scoreboard styling with translucent backgrounds

### Fixed
- Title no longer gets cut off on smaller screens
- Improved responsive design for mobile devices

## [1.0.0] - 2026-07-19

### Added
- Initial release of Pong Game
- Classic Pong gameplay with player vs AI
- Mouse and keyboard controls (Arrow Keys, W/S)
- Physics-based ball movement with collision detection
- Paddle collision with spin system
- Wall collision detection
- Scoreboard with first-to-5 win condition
- Pause/Resume functionality
- Score reset button
- Game over alerts

### Features
- HTML5 Canvas-based rendering
- Vanilla JavaScript (no dependencies)
- 800x400 pixel game area
- 60 FPS gameplay using requestAnimationFrame
- Green paddles and red ball (original color scheme)
- Center dividing line
